## All Valid Tibetan Stacks

### Superscripts (ra-mgo, la-mgo, sa-mgo)

**ra-mgo (ར) stacks:**
```
རྐ    རྒ    རྔ    རྗ    རྙ    རྟ    རྡ    རྣ    རྦ    རྨ    རྩ    རྫ
```

**la-mgo (ལ) stacks:**
```
ལྐ    ལྒ    ལྔ    ལྕ    ལྗ    ལྟ    ལྡ    ལྤ    ལྦ    ལྷ
```

**sa-mgo (ས) stacks:**
```
སྐ    སྒ    སྔ    སྙ    སྟ    སྡ    སྣ    སྤ    སྦ    སྨ    སྩ
```

### Subscripts

**wa-zur (ྭ) subscripts:**
```
ཀྭ    ཁྭ    གྭ    ཅྭ    ཉྭ    ཏྭ    དྭ    ཙྭ    ཚྭ    ཞྭ    ཟྭ    རྭ    ཤྭ    སྭ    ཧྭ
```

**ya-btags (ྱ) subscripts:**
```
ཀྱ    ཁྱ    གྱ    པྱ    ཕྱ    བྱ    མྱ
```

**ra-btags (ྲ) subscripts:**
```
ཀྲ    ཁྲ    གྲ    ཏྲ    ཐྲ    དྲ    པྲ    ཕྲ    བྲ    མྲ    ཤྲ    སྲ    ཧྲ
```

**la-btags (ླ) subscripts:**
```
ཀླ    གླ    བླ    ཟླ    རླ    སླ
```

### Complex Stacks (Multiple components)

**Superscript + Subscript combinations:**
```
རྐྱ    རྒྱ    རྨྱ    རྒྭ    རྩྭ
སྐྱ    སྒྱ    སྤྱ    སྦྱ    སྨྱ
སྐྲ    སྒྲ    སྣྲ    སྤྲ    སྦྲ    སྨྲ
གྲྭ    དྲྭ    ཕྱྭ
```
